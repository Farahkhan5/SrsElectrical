# 🔌 Lab Automation System

## 📌 Overview
Lab Automation System ek web-based application hai jo electrical lab ke environment ke liye banaya gaya hai. Is system me fuse, capacitor aur other electrical components ki testing aur record management ki jati hai. Har test ke liye system automatically ek **unique ID (10–12 digits)** generate karta hai taake har record ko easily track aur identify kiya ja sake.

---

## ⚙️ Features
- Fuse, Capacitor aur other components ki testing management  
- Automatic Unique ID generation (10–12 digits)  
- Complete test records ka storage system  
- Records ko search aur track karne ki facility  
- Simple aur user-friendly interface  
- Secure database handling  

---

## 🛠️ Technologies Used
- PHP (Core PHP)
- MySQL
- HTML5
- CSS3
- JavaScript

---

## 📂 Project Structure
- `/includes` – Database connection and reusable files  
- `/assets` – CSS, JS, and images  
- `/pages` – Main application pages  
- `/database` – SQL structure and tables  
- `index.php` – Main entry point

----

## 🔑 Unique ID System
Each test entry generates a unique 10–12 digit ID automatically to ensure:
- No duplication of records
- Easy tracking of test history
- Secure identification of components
